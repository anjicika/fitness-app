## Navodila in pomoč
### 1. Pred začetkom dela
Vedno posodobi lokalni repozitorij, da imaš najnovejše spremembe:
```bash
git pull
```
### 2. Dodaj in commitaj svoje spremembe
```bash
git add .
git commit -m "Kratek opis spremembe"
```
### 3. Na koncu dela pushaj
```bash
git push
```
